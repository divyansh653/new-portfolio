All HTML Tags and CSS Properties with Examples

Introduction

This documentation provides an extensive overview of HTML tags and CSS properties with explanations and examples to help developers understand how to create, style, and manipulate web content effectively.

HTML Tags

HTML (Hypertext Markup Language) uses tags to structure the content of web pages. Below is a list of the most commonly used HTML tags and their descriptions:

Basic HTML Tags

<!DOCTYPE html>  
<html>  
<head>  
<title>  
<body>  
<h1> to <h6>  
<p>  
<a>  
<img>  
<ul>, <ol>, <li>  
<div>  
<span>  
<form>  
<input>  
<button>  
<footer>  
<header>  
<article>  
<section>  
<aside>  
<nav>

Attributes of HTML Tags

HTML tags can have attributes that provide additional information or control their behavior. For example:

<a href="https://example.com">Click here</a>  
<img src="image.jpg" alt="Image description">

CSS Properties

CSS (Cascading Style Sheets) is used to style the HTML elements. Below is a comprehensive list of CSS properties with examples:

General Styling Properties

color: #333;  
background-color: #fff;  
font-family: Arial, sans-serif;  
font-size: 16px;  
line-height: 1.5;  
text-align: center;  
text-decoration: underline;  
font-weight: bold;  
font-style: italic;

Box Model Properties

The CSS Box Model defines the layout of elements. It consists of the content area, padding, border, and margin:

width: 200px;  
height: 100px;  
padding: 10px;  
border: 1px solid #ccc;  
margin: 20px;  
box-sizing: border-box;

Text and Font Properties

font-family: "Helvetica", sans-serif;  
font-size: 18px;  
text-transform: uppercase;  
letter-spacing: 1px;  
text-shadow: 2px 2px 4px #000000;

Layout Properties

Layout properties control the position and arrangement of elements on the page:

display: block;  
display: inline;  
display: flex;  
position: absolute;  
top: 20px;  
left: 50px;  
z-index: 10;

Background and Borders

background-color: #3498db;  
background-image: url('background.jpg');  
background-size: cover;  
border-radius: 10px;  
border-width: 2px;  
border-style: solid;

Flexbox Properties

Flexbox is used for aligning and distributing elements within a container:

display: flex;  
justify-content: space-between;  
align-items: center;  
flex-wrap: wrap;

Grid Properties

CSS Grid is a two-dimensional layout system:

display: grid;  
grid-template-columns: 1fr 1fr 1fr;  
grid-template-rows: auto;  
gap: 20px;

Responsive Design (Media Queries)

Media queries allow CSS styles to be applied based on the screen size or device type:

@media (max-width: 768px) {
    body {
        background-color: lightblue;
    }

    h1 {
        font-size: 24px;
    }
}
